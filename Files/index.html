Online Image Translator

Select your action:
  • Translate a whole image
  • Scan and translate a region
  • Settings

By default, it connects to an ImageTrans server instance which is configured to translate Japanese manga. You can configure its URL in settings. If you do not have ImageTrans, please go to settings to select Tesseract-OCR or PaddleOCR, which works completely on your browser.

Check out the gallery to see some translated images samples.
